# MeeAir

Веб-приложение для поиска и бронирования авиабилетов.  
Разработано в рамках дипломного проекта (Колледж КМЭПТ, Москва, 2026).

## Основные возможности

- Поиск рейсов по направлениям с фильтрацией  
- Пошаговое бронирование: класс обслуживания, выбор места, дополнительные услуги  
- Личный кабинет с историей бронирований  
- Редактирование и отмена билета (до 12 часов до вылета)  
- Подтверждение email при регистрации (6-значный код)  
- Административная панель: управление рейсами, аэропортами, пользователями, курсами валют  
- Мультивалютность (EUR, USD, RUB)  
- Полная адаптация под мобильные устройства, планшеты и ПК  
- Современный дизайн и анимации

## Технологии

- **Backend:** Python, Flask, Flask-Login, Werkzeug  
- **Database:** SQLite, SQLAlchemy ORM  
- **Frontend:** HTML5, CSS3, JavaScript (нативный), Google Fonts  
- **Безопасность:** хеширование паролей, защита сессий, валидация email

## 📝 Список изменений

### v1.2.0 – Текущая версия

### Баги:
- Кнопка «Выбрать»:
Исправлен openBooking: корректные города, проверка мест, загрузка занятых мест. Телефон подставляется из профиля.
- Админ-вкладки на телефоне:
Вкладки обёрнуты в .admin-tabs-scroll с горизонтальной прокруткой на узких экранах. Исправлен ошибочный тег <motion>.
- «Прыгающий» шрифт:
Время рейсов и цифры переведены на Montserrat с tabular-nums и ровным line-height. Syne убран полностью.
- Кнопка «Главная»:
Ссылки с href="#" и preventDefault, при повторном нажатии на главной — плавный скролл наверх.

### Доработки:
- Кастомизация профиля:
Выбор цвета аватара (6 цветов).
- Эмодзи и картинки:
Эмодзи убраны с сайта (кроме админки: ✏️ / 🗑️). На главной — SVG-иконки услуг и блока преимуществ. В модалке бронирования — те же картинки / текстовые бейджи Wi‑Fi и VIP.

### Бэкенд (app.py, models.py, seed.py)
- Бронирование: проверка суммы на сервере, занятости места и лимита мест по классу, поля passengers и return_date.
- API: /api/flights/<id>/seats — занятые места; в поиске — города и список taken_seats.
- Авторизация: вход только после подтверждения email (кроме админа).
- Админка: редактирование аэропортов (PUT), исправлено дублирование при создании брони.
- Миграция БД: при python seed.py или запуске app.py добавляются колонки passengers и return_date.

### Фронтенд (app.js, index.html, style.css)
- Поиск: дата вылета/возврата, число пассажиров, режим «туда-обратно», индикатор «нет мест».
- Бронирование: загрузка занятых мест, валидация карты (демо), учёт пассажиров в цене, состояние загрузки при оплате.
- Мои билеты: полные карточки с услугами, датой, кнопками редактирования/отмены.
- Редактирование услуг: корректные ключи (bag, ins, …), loadRatesUI, updateEditTotal.
- Визуал: шрифт Syne, эмодзи вместо отсутствующих картинок, стили билетов, пустых состояний, проданных рейсов.
- Прокрутка: убран перехват колеса мыши — обычная плавная прокрутка.
